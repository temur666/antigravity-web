# CSS 架构重构 - 单文件拆分为组件级文件

## 概述

将 3018 行的巨型 `index.css` 拆分为 13 个组件级 CSS 文件，`index.css` 精简至约 160 行（仅保留全局变量、Reset、keyframes、骨架布局）。

## 变更文件

### 新增 CSS 文件（12 个）

| 文件 | 职责 |
|------|------|
| `components/Sidebar/Sidebar.css` | 侧边栏 + 移动端适配 |
| `components/ChatPanel/ChatPanel.css` | 聊天面板布局 + 翻页模式 + 空状态 |
| `components/ChatPanel/Steps.css` | 所有 Step 类型样式 + AI 响应排版 |
| `components/ChatPanel/InputBox.css` | 输入框 + 拖拽 + 附件 + dropdowns |
| `components/ChatPanel/MetadataPopover.css` | 元数据弹出层 |
| `components/Header/Header.css` | 顶栏 + 下拉菜单 + 模型选择器 |
| `components/ConfigPanel/ConfigPanel.css` | 配置面板 + 状态指示器 |
| `components/Dashboard/Dashboard.css` | 仪表盘 + 任务卡片 |
| `components/BottomNav/BottomNav.css` | 底部导航栏 |
| `components/NotesPage/NotesPage.css` | 笔记编辑器 |
| `components/BottomSheet.css` | 底部抽屉 |
| `components/InstallPrompt/InstallPrompt.css` | PWA 安装提示 |

### 修改文件

- `src/index.css` — 精简至全局样式（变量、Reset、共享 keyframes）
- 12 个 `.tsx` 文件 — 各自添加 `import './Xxx.css'` 语句

## 设计决策

- **不使用 CSS Modules**：避免 335 个 className 引用重写
- **移动端 `@media` 随组件走**：每个组件文件末尾包含自己的响应式规则
- **跨组件选择器**：保留在 index.css 或最相关的组件文件中
