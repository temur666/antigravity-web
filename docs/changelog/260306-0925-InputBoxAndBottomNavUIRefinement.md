# Changelog: InputBox and BottomNav UI Refinement

## Changes
- Removed top borders from `.input-box` and `.bottom-nav` to eliminate visual segmentation.
- Removed borders and background from `.input-box-inner-row` to fuse the input field seamlessly with the bottom area.
- Added a soft, fading `box-shadow` to `.input-box` to blend perfectly with the chat background (`var(--color-bg-base)`) and maintain a smooth reading fade.
- The bottom UI is now immersive and visually integrated, inspired by modern music player aesthetics.
